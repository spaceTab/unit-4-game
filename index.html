Crystall Collecter

At the start of each game the user is given a random sum

As well each crystal is assigned a random number

the value you of the gem will then be added on each click event

the objective is for the user to click the gems, and gain the same

sum as the randomly given sum at the start of each game

Otherwise if the sum of gems clicked goes over the random sum

the users looses
